model
temperature這個參數控制生成文字的隨機性。數值範圍是 0 到 1。值越高（接近 1），生成的文字越隨機和創意；值越低（接近 0），生成的文字越保守和精確。
max_tokens這個參數決定生成回應的最大字數。你設置的 256 表示回應最多可以包含 256 個 token（包括字詞和標點符號）。
top_p這個參數控制生成文字時的隨機性。這是一個累積概率的截斷值。設定為 1 表示不使用這個截斷技術，允許模型考慮所有可能的下一個 token。
frequency_penalty這個參數控制模型對於重複用詞的懲罰程度。範圍是 -2.0 到 2.0。較高的值會使模型更少重複用詞，較低的值則反之。你設置的 0 表示不對重複用詞進行懲罰。
presence_penalty這個參數控制模型對於引入新話題或詞彙的懲罰程度。範圍是 -2.0 到 2.0。較高的值會使模型更傾向於引入新話題，較低的值則反之。你設置的 0 表示不對引入新話題進行懲罰。
